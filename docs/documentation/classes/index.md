## Basic Test Structure

Tests for classes should be placed in files under `spec/classes/`. For
example the tests for `apache::install` should be in
`spec/classes/apache_install_spec.rb`.

{% highlight ruby %}
require 'spec_helper'

describe '<class name>' do
  # let(:params) { ... }

  # it { is_expected.to ... }
end
{% endhighlight %}
